## Task1

### Python Operations Used
- Iterating, e.g `for v in l:`
    - Complexity: O(n)
- Inclusion, e.g `x in/not in l`
    - Complexity: O(n)
- Indexing, e.g `l[i]`
    - Complexity: O(1)
- Appending, e.g `l.append(1)`
    - Complexity: O(1)

### Complexity
Time Complexity: O(n^2)
Space Complexity: O(n)
